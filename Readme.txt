Folder server คือระบบหลังบ้าน
Folder client คือระบบหน้าบ้าน
ไฟล์ project_db.sql คือฐานข้อมูล

คู่มือการติดตั้งโปรแกรมระบบซื้อขายและแลกเปลี่ยนโบราณวัติถุและของสะสมมีค่า
	หลังจากติดตั้ง VSCode MAMP Node.js แล้วให้เรามาทำตามขั้นตอนดังนี้เพื่อเปิดโปรแกรมระบบซื้อขายและแลกเปลี่ยนโบราณวัติถุและของสะสมมีค่า สามารถทำตามขั้นตอนได้ดังนี้
	
1 การดาวน์โหลดโปรแกรมจาก Github
	ไปที่ เว็บไซต์ https://github.com/HomEza204/Project-Web.git
	ทำตามขั้นตอนดังต่อไปนี้
•	คลิก “Code” 
•	หลังจากนั้นให้ คลิก “Download ZIP”
•	เลือกโฟลเดอร์ที่ต้องการติดตั้ง
•	คลิก “Save”
	แตกไฟล์ ZIP

2 วิธีติดตั้งฐานข้อมูล
	เปิดโปรแกรม MAMP
	คลิก “Open WebStart page”
	เลือกไปที่ “TOOLS” แล้วคลิก “PHPMYADMIN”
	คลิก “new” แล้วทำตามขั้นตอนดังนี้
•	ช่อง Database name ให้ใส่ “project_db”
•	อีกช่องให้เปลี่ยนเป็น “utf8_general_ci”
•	คลิก “Create”
	ขั้นตอนการ import ทำตามคำแนะนำดังนี้
•	คลิกฐานข้อมูล “project_db”
•	เลือก “Import”
•	คลิก “Choose File”
•	เลือก “project_db.sql” จากโฟลเดอร์ที่เราติดตั้งโปรแกรมจาก Github
•	คลิก “Go”

3 วิธีเปิดโปรแกรม
	เปิดโปรแกรม VSCode
	เลือกไปที่ “File” หลังจากนั้นคลิก “Add Folder to Workspace…”
•	เลือกโฟลเดอร์ที่ติดตั้งจาก Github
	การรันโปรแกรมจากฝั่ง Server
•	เลือกไปที่ Terminal
•	หลังจากนั้นคลิก “New Terminal”
•	เลือกโฟลเดอร์โปรเจคของเรา
•	ใน่ช่อง Terminal ให้พิมพ์ “cd server”
•	หลังจากนั้นให้พิมพ์ “npm i” เพื่อติดตั้ง Dependencies
•	หลังจากนั้นให้พิมพ์ “npm run dev” ในช่อง Terminal เพื่อเปิดโปรแกรม
	การรันโปรแกรมจากฝั่ง Client 
•	เลือกไปที่ Terminal
•	หลังจากนั้นคลิก “New Terminal”
•	เลือกโฟลเดอร์โปรเจคของเรา
•	ใน่ช่อง Terminal ให้พิมพ์ “cd client”
•	หลังจากนั้นให้พิมพ์ “npm i” เพื่อติดตั้ง Dependencies
•	หลังจากนั้นให้พิมพ์ “npm start” ในช่อง Terminal เพื่อเปิดโปรแกรม
